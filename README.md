# outdoor-gear-recommender

## 项目简介

**基本介绍**

户外运动装备智能推荐系统面向徒步、露营、登山等户外爱好者，提供装备浏览、出行计划管理与个性化推荐服务。用户可创建出行计划并填写目的地、季节、人数、预算及自由文本需求，系统结合用户偏好与计划约束，为其推荐帐篷、睡袋、背包、服装等装备；同时支持管理端维护装备知识库与用户权限。技术上采用 Spring Boot 4 + Vue 3 前后端分离架构，数据层使用 MySQL 与 Redis。

**项目特点**

- **混合推荐算法**：约束过滤（季节、人数、预算等）→ 协同过滤（相似用户/计划选品）→ 内容推荐（标签与用户画像匹配），并做品类多样性控制，避免推荐结果单一。
- **计划自由文本理解**：支持在创建计划时输入一段需求描述，系统自动抽取关键词并推断季节、活动类型，用于更精准的推荐匹配。
- **多维度用户画像**：除季节、活动类型、预算外，支持偏好目的地、装备分类、标签及难度偏好（入门/进阶/专业），用于冷启动与无计划场景的个性化推荐。
- **真实装备数据**：内置 1000+ 真实可查装备及变体，覆盖帐篷、睡袋、背包、冲锋衣等品类，支持舒适温标、场景、季节等属性，便于推荐与展示。

## 技术栈

- 后端：Spring Boot 4、Spring Security + JWT、Spring Data JPA、MySQL、Redis
- 前端：Vue 3 + Vite + Element Plus（已实现用户端与管理端核心页面）

## 快速开始

1. 启动依赖：`docker compose up -d`（MySQL + Redis）
2. 启动后端：`mvn spring-boot:run -Dspring-boot.run.profiles=local`
3. 启动前端：`cd frontend && npm install && npm run dev`
4. 详见 [docs/快速开始.md](docs/快速开始.md)
