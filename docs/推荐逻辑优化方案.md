# 推荐逻辑优化方案（不修改代码版）

## 一、当前系统能力诊断

### 已有能力（实际存在）

| 模块 | 功能 | 现状 |
|------|------|------|
| **约束过滤** | 季节、预算、场景、人数 | ✅ 生效，使用 `LIKE` 模糊匹配 |
| **内容匹配** | 标签 Jaccard 相似度、季节/场景包含 | ✅ 生效，但计划无季节/场景时内容分为空 |
| **协同过滤** | 基于用户行为的相似用户推荐 | ⚠️ 冷启动下几乎无数据，得分常为 0 |
| **分类多样性** | 轮流从各类选取 | ✅ 已实现 |

### 感知为「只有约束+关键字」的原因

1. **计划信息不完整**：若计划未填季节、活动类型，内容匹配无输入，主要靠约束过滤
2. **协同过滤冷启动**：用户少、行为少时，协同得分为 0，相当于未生效
3. **标签映射单一**：计划活动类型只映射到「名称包含」的标签，覆盖有限

## 二、不修改代码可做的优化

### 1. 完善出行计划数据（最重要）

创建/编辑计划时，尽量填全以下字段：

| 字段 | 作用 | 建议 |
|------|------|------|
| **季节** | 约束过滤 + 内容匹配 | 必填：春/夏/秋/冬/四季 |
| **活动类型** | 约束过滤 + 内容匹配 | 必填：徒步、露营、登山、自驾露营等 |
| **预算** | 约束过滤 | 填总预算，系统会按 1.2 倍算单件上限 |
| **人数** | 约束过滤 + 内容匹配 | 填出行人数 |

**示例**：泰山一日游 → 季节「春」或「秋」，活动类型「徒步」或「登山」，预算 500–1000，人数 2。

### 2. 装备数据质量（通过管理端）

在管理端装备管理中：

- **季节**：填「春,夏,秋」或「四季」，便于与计划匹配
- **场景**：填「徒步,露营」或「徒步,登山,露营」，与活动类型对应
- **标签**：多选与装备特性相关的标签（轻量化、防水、徒步、露营等）

标签越丰富，Jaccard 相似度越有区分度。

### 3. 积累用户行为数据（激活协同过滤）

协同过滤依赖收藏、评分、加购、购买等行为：

- 用测试账号多做一些收藏、加购、评分
- 不同用户对同一装备有相似行为时，协同过滤才会产生有效得分

**已自动初始化**：`TestBehaviorDataInitializer` 在 local 环境下会创建：
- 测试用户：user1 / user2 / user3 / user4 / user5（密码均为 123456）
- 收藏、加购、评分数据，且用户间有重叠（便于协同过滤找相似用户）

首次启动或清空数据库后，行为数据会自动初始化。

### 4. 用户偏好设置（个人中心）

在个人中心填写「推荐偏好」：

- 偏好季节、活动类型、预算区间、偏好目的地、偏好装备分类、偏好标签、难度偏好（入门/进阶/专业）
- 无出行计划时，系统会用这些偏好做内容匹配，缓解冷启动

## 三、推荐效果自检清单

| 检查项 | 操作 |
|--------|------|
| 计划是否填了季节、活动类型 | 编辑计划，补全字段 |
| 计划预算是否合理 | 如 500–2000，避免过小导致候选为空 |
| 装备是否有标签 | 管理端查看，重点装备补标签 |
| 是否有用户行为数据 | 多账号收藏/加购，观察推荐是否变化 |
| 是否重启过后端 | 确保分类多样性等逻辑已生效 |

## 四、已实现的代码级优化

| 方向 | 说明 |
|------|------|
| **活动类型→标签映射** | 已实现：`ActivityTypeTagMapping` 提供「徒步→[徒步,轻量化,便携]」「露营→[露营,防水,防风]」等映射，内容层在计划/用户偏好的活动类型下会扩展匹配这些标签。 |
| **冷启动兜底** | 已实现：当协同+内容得分均为 0 时，按「评分条数」给候选装备加热门加分；若推荐条数不足 topN，用热门装备（按评分条数排序、上架且未删除）补足。 |

## 五、后续可考虑的代码级优化（供实现时参考）

| 方向 | 说明 |
|------|------|
| **目的地/难度参与匹配** | 目的地、难度等级参与约束或内容得分 |
| **权重可配置** | 协同/内容/基础分权重可配置，便于调参 |
| **TF-IDF 或 BM25** | 用文本相似度替代简单包含，提升语义匹配 |
| **负反馈** | 用户「不感兴趣」等行为参与过滤或降权 |

---

## 六、总结

**不修改代码时，优先做三件事：**

1. **计划必填季节、活动类型**，让内容匹配有输入
2. **通过管理端补全装备标签**，提升 Jaccard 区分度
3. **用多账号积累收藏/加购**，激活协同过滤

这三步完成后，推荐会从「主要靠约束过滤」变为「约束 + 内容 + 协同」共同作用，效果会有明显提升。
